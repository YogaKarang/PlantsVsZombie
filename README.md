# PlantsVsZombie
Game Plant vs Zombie merupakan game strategi di mana pemain harus menempatkan berbagai jenis tanaman untuk menghadapi serangan zombie yang datang secara bertahap. Setiap tanaman memiliki kemampuan yang berbeda, sehingga pemain perlu menyusun strategi agar dapat bertahan dan memenangkan permainan.
